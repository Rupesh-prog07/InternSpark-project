# Deep Learning Image Classification

## Project Overview
This project implements an image classification model using Deep Learning. The notebook covers data preprocessing, model creation, training, evaluation, and prediction.

## Repository Structure
```
Task-2-Deep-Learning-Image-Classifier/
│── deep-learning-image-classifier.ipynb
│── README.md
│── requirements.txt
│── .gitignore
│── dataset/
│── models/
│── screenshots/
│── docs/
```

## Environment Setup

```bash
python -m venv venv
# Windows
venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
jupyter notebook
```

Open `deep-learning-image-classifier.ipynb` and run all cells.

## Requirements
- Python 3.10+
- tensorflow
- numpy
- pandas
- matplotlib
- scikit-learn
- jupyter
