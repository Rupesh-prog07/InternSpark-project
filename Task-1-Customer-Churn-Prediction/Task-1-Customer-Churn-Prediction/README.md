# Customer Churn Prediction using Machine Learning

## Overview
This project predicts whether a telecom customer will churn using supervised machine learning.

## Files
- Customer-Churn-Corrected.ipynb
- dataset/Customer-Churn.xls
- requirements.txt
- docs/Task-1-Report.pdf

## Environment Setup

```bash
python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
jupyter notebook
```

Open `Customer-Churn-Corrected.ipynb` and run all cells.

## Requirements

- Python 3.10+
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- jupyter
